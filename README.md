# 🦾 WebTextScraper (WebPaid Bypass)


This project is created for the purpose of researching web security techniques and demonstrating the ability to extract data from multi-layered content locking systems.
This project is created for people who like to read books offline, some books only available on paid websites. Install Tampermonkey extension, save the script and you'll get raw .txt file of the locked content 
## 🛠️ Features
- **Hybrid Scraper:** Automatically detects Free and VIP chapters.
- **Deep Source Inspection:** Uses Regex to extract hidden data within massive JSON strings embedded in HTML source code.
- **Auto-Navigation:** Automatically switches chapters and stores data in LocalStorage.
- **Data Reconstruction:** Restores text formatting from encoded Unicode characters.

## 🛡️ Security Layers Bypassed
1. **Server-side Truncation:** Overcame the 500-character return limit.
2. **JSON Obfuscation:** Located and decoded hidden data string.
3. **Hydration Protection:** Captured data before the system could clear memory.

## 🚀 Usage
1. Install the Tampermonkey extension.
2. Copy the source code from `Data_Scraper.user.js` into a new script.
3. Access Chapter 1 and let the system run automatically.
4. Type `downloadFile()` in the Console to get the full TXT file.
